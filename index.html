序

王小波曾在《万寿寺》的结尾中写道：「一切都在无可挽回地走向庸俗」。然而，就在同一段中，他却也写了下那句：「一个人只拥有此生此世是不够的，他还应该拥有诗意的世界」。正如他在同一段展现的不同句子一样，终点也许是让人失望的，但在路上，还是要好好走，「永远青春，永远热泪盈眶」。

《叫魂：1768年中国妖术大恐慌》读后感

发表于 2020-03-29 更新于 2020-07-11 分类于 大学

「1768年，中国悲剧性近代的前夜。」——这是全书的第一句。

其时正是所谓「康乾盛世」，表面光鲜亮丽，人口、商贸、农业等等似乎都处在一个顶峰期；然而「叫魂案」，这个神奇的幽灵，就在这「镀金时代」，游荡在清代中国大地上。它以荒谬的方式出现，又以荒谬的方式结束。

而孔飞力先生的《叫魂：1768年中国妖术大恐慌》一书，就旨在通过描述与分析这场全社会的叫魂恐惧，借此更深入地理解那个时代及其背后的历史脉络。

冷眼看军训

发表于 2019-08-27 更新于 2020-07-11 分类于 大学

一个认同王小波的人，不会认同军训隐含的价值观念。

尤瓦尔·赫拉利在《人类简史》提过：

在基因、荷尔蒙、有机体之外，人类社会的另外一条重要主线，这就是意识形态、想象力和讲故事的能力。正因为语言的出现和讲故事能力的提升，人们能够以极其灵活的方式与陌生人进行大规模的协作。

是的，历史的进程在此角度上就是人类想象力的变化演进。奴隶，封建，王朝，民主，党派，战争，经济，商业……都是人类想象力的结果，是人类设想的故事与情境。

而进入浙江大学后的军训，就是要让我们模拟进入人类想象力具象化后的国家暴力机构——军队——的生活情景。

若是你愿意彻头彻尾融入这个军训的故事，成为军队中的一颗螺丝钉，一株韭菜，一个演员，那么请不要看这篇文章，我尊重你，但我并不认同你。

又是一年过去，绚烂的烟花把漆黑的天空照亮得泛白，轰隆声不断，耳膜被震得颤动起来，眼前的鞭炮放完后留下浓郁的烟雾……

生活是流动的盛宴。这盛宴的餐桌是流动的时间之河，从一切之无流到此处，从此处流向彼处，直至流向无限、而我，我不是个整体，我是散落于河流上的碎片，是摆放在餐桌上的菜肴，这半桌是我的内在，那半桌是我外在的呈现，这一小盘是我的知识储备，那一小盘是我的憎恶之物……这餐桌流动着，我的各个部分也在不断流动重组。“人永远无法踏进同一条河流。”在时间的驱使下，这一刻的我已与上一刻的我在有些地方不同了。这就是流动的盛宴，而命运，就是这场盛宴的饕餮，从人之初贪婪又慢条斯理地品味到生命的最后一刻。
