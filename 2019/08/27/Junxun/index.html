冷眼看军训

发表于 2019-08-27 更新于 2020-07-11 分类于 大学 Valine：

一个认同王小波的人，不会认同军训隐含的价值观念。

尤瓦尔·赫拉利在《人类简史》提过：

在基因、荷尔蒙、有机体之外，人类社会的另外一条重要主线，这就是意识形态、想象力和讲故事的能力。正因为语言的出现和讲故事能力的提升，人们能够以极其灵活的方式与陌生人进行大规模的协作。

是的，历史的进程在此角度上就是人类想象力的变化演进。奴隶，封建，王朝，民主，党派，战争，经济，商业……都是人类想象力的结果，是人类设想的故事与情境。

而进入浙江大学后的军训，就是要让我们模拟进入人类想象力具象化后的国家暴力机构——军队——的生活情景。

若是你愿意彻头彻尾融入这个军训的故事，成为军队中的一颗螺丝钉，一株韭菜，一个演员，那么请不要看这篇文章，我尊重你，但我并不认同你。

私以为，任何一个拥有自由意志与独立人格的人，都应该或明确或模糊地有自己的三观了（这是专属于你一人的小世界）。其中自然有许多规则，有些规则应该是被公认且信奉为真理的，比如自由与平等，比如独立思考的能力。

引用美国「独立宣言」中的话。

我们认为这些真理是不言而喻的：

人人生而平等，造物者赋予他们若干不可剥夺的权利，其中包括生命权、自由权和追求幸福的权利。

然而，参与军训的我们或自愿或被迫地放弃自己的自由权，放弃自己的平等权，放弃自己独立思考的能力，放弃伸张正义与辩论真理的权利。

多说句话就是对教官有意见，就是反抗，就要惩罚。所谓的事后讲理也不过是个客套话。

莫名其妙教官就可以找个由头来训斥你，惩戒你，罚你深蹲，罚你站军姿，喊报告，罚你冲刺跑，俯卧撑，高抬腿……

话语权掌握在教官那里，所谓言出即法，他说的话就构成了一个粗糙的权力体系。这个体系很不完整且没有边际，所以权力的大小没有定数，随机可变，一切跟随教官的兴致和意愿。教官开心满意了，也许就没有惩罚，可以早点休息；教官若是不开心，那你就要成为他的消遣，用你遭受的惩罚来给他快感，让他高兴。

一个以教官为主体的威权主义，一个以「服从」为天命、以「受苦」为享乐的畸形的装置。

人的个体性消泯，人的主体性丧失，人沦落为一个庞大而抽象的整体中的一个不起眼的小点，一个淹没在以集体主义为名义实行权威的沼泽中的溺者。

人性，人性。

军训就是反人性化，去人性化的。

一切都是朝着非人性化、机器化、归顺化的方向蔓延，反过来要吞噬你的个性，吞噬你的思想，吞噬你的自由意志与独立人格。

最愿意你成为一头蠢驴，听话，听话就好。

然而，这与我「让自由之风长存于地」的信念完全相悖。

我生活在美与自由的想象世界中，而军训却要让我进到那个纪律与惩戒的想象世界中。

当你真的被「洗脑」了而愿意完全接纳那个想象世界的时候，你就会以那个世界中的价值观念为导向。军训中，我们以听教官话，接受惩罚，参与军训的评价体系与竞争系统为准则，然后各连之间陷入一个恶性竞争的泥沼之中。不禁想起高考，一个以分数高低为评价体系的想象世界，一切都以考试为规、以分数为矩，个体的创造性与独特性同样泯灭。

于军训中的价值体系中，我们做的一切似乎都很合理，都理所当然；然而以局外人的视角观看，一切又是那么反人性，那么滑稽，那么荒诞。

也许有人说军训了之后让你看看，这个世界还是有许多残酷而艰辛生活着的地方的，你要好好珍惜眼前的美好大学时光；我却反着想，军训让你看到了原来世界上还有这么反人性而荒谬的存在，让你知道，我们还需要去做很多很多才能改变世界，真正地独自自在地生活，栖息于大地之上。

又有人说，以后回忆军训时总是明媚的色彩。然而这不过是因为记忆会粉饰生活，你的记忆总是会倾向摸出不好的阴影而去美化你的经历，只留下美好的事情。我们要做的不是忘却，而是记住，要以后去改变这种及其荒谬违背人性的事情。

最后，引一段《庆余年》的话。

我希望庆国的人民都能成为不羁之民。受到他人虐待时有不屈服之心，受到灾恶侵袭时有不受挫折之心，若有不正之事时，不恐惧修正之心，不向豺虎献媚……我希望庆国的国民，每一位都能成为王，都能成为统治被称为“自己”这块领土的……独一无二的陛下，我的王。

——叶轻眉

是日夜，愤而记
